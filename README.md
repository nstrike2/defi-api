# defi-api

## Getting Started

```sh
git clone https://github.com/nstrike2/defi-api.git
yarn install
```

## Development

**fe**
```sh
yarn client
```

**be**
```sh
yarn server
```

**blockchain**
```sh
yarn axelnet-blockchain
```

**fe, be, & blockchain**
```sh
yarn start
```

**load axelnet wallet**
```sh
yarn load-axelnet-wallet
```
